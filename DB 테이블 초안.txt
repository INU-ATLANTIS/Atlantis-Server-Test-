사용자(User)
ID: 고유 식별자 (PK)
별명: 사용자 별명
이메일: 사용자 이메일 (유니크)
비밀번호_해시: 비밀번호 저장을 위한 해시 값
생성_날짜: 계정 생성 날짜
업데이트_날짜: 정보 최종 업데이트 날짜

게시글(Post)
ID: 고유 식별자 (PK)
사용자_ID: 게시글을 작성한 사용자의 ID (FK)
제목: 게시글의 제목
내용: 게시글의 본문 내용
추천수: 게시글 추천수
댓글수: 댓글 갯수
생성_날짜: 게시글 작성 날짜
업데이트_날짜: 게시글 업데이트 날짜
마커_ID: 게시글과 연결된 마커 정보의 ID (FK, NULL 가능) // NULL인 경우 마커 정보 없이 게시글만 존재

댓글(Comment)
ID: 고유 식별자 (PK)
게시글_ID: 댓글이 속한 게시글의 ID (FK)
사용자_ID: 댓글을 작성한 사용자의 ID (FK)
내용: 댓글의 본문 내용
추천수: 댓글의 추천수
부모_댓글_ID: 대댓글 기능을 위한 부모 댓글의 ID (FK, NULL 가능). // 이 필드는 댓글이 다른 댓글에 대한 답변인 경우 해당 댓글의 ID를 참조합니다. 최상위 댓글의 경우 NULL입니다.
생성_날짜: 댓글 작성 날짜
업데이트_날짜: 댓글 최종 업데이트 날짜

알림(Notification)
ID: 고유 식별자 (PK)
타입: 알림 유형 (예: 위치 근접, 시간 기반 등)
대상_사용자_ID: 알림을 받을 대상 사용자의 ID (FK, NULL 가능) //NULL인 경우 강제 알림
메시지: 알림 메시지
생성_날짜: 알림 생성 날짜

권한(Permission)
ID: 고유 식별자 (PK)
사용자_ID: 권한이 부여된 사용자의 ID (FK)
권한_유형: 권한의 종류 (예: 마커 생성, 알림 전송)

마커(Marker)
ID: 고유 식별자 (PK)
위도: 마커의 위도
경도: 마커의 경도
사용자_ID: 마커를 생성한 사용자의 ID (FK)
설명: 마커에 대한 설명
생성_날짜: 마커 생성 날짜

건물(Building)
ID: 고유 식별자 (PK)
이름: 건물 이름
위치_ID: 건물의 마커 ID (FK)

평면도(FloorPlan)
ID: 고유 식별자 (PK)
건물_ID: 평면도가 속한 건물의 ID (FK)
층: 건물 내의 층
이미지_URL: 평면도 이미지 경로


============보류================

사용자_즐겨찾기(User_Favorites)
ID: 고유 식별자 (PK)
사용자_ID: 즐겨찾기를 설정한 사용자의 ID (FK)
강의실_ID: 즐겨찾는 강의실의 ID (FK)

강의실(Classroom)
ID: 고유 식별자 (PK)
건물_ID: 강의실이 위치한 건물의 ID (FK)
층: 강의실이 위치한 층
번호: 강의실 번호
